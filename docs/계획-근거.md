# 계획의 근거 — 왜 이 앱인가

app-radar 시장조사에서 도출. 여러 후보(차계부·독서기록·체중기록 등)가 "국산 1인 강자 존재"로 탈락한 끝에, 온욕(사우나·찜질방·온천)만 **검증된 모델 + 빈자리 + 데이터 + 수요**가 모두 성립.

## 1. 벤치마크: サウナイキタイ가 이긴 구조
월 120만 이용 · 게재 15,000곳 · 방문기록(サ活) 누적 880만.
- **① 초정밀 시설 스펙 DB** — 온도/로우류/외기욕/지하수/타투OK 등 100+ 필터
- **② 방문기록·후기 UGC 커뮤니티(サ活)** ← 핵심 엔진, 네트워크 효과
- 인기순·혼잡도·여성전용·영업중·숙박가능 필터
- 약점: 앱 크래시·로그인 버그(불만 대다수 안정성), "구글맵으로도 됨"
- **한국 접근 불가**: iOS는 한국 스토어에 없음, Play는 지역잠금. 콘텐츠도 100% 일본. → 원조가 한국에 직접 진출할 유인 낮음 = 지리적 이식 기회.

## 2. 국내 경쟁 = 사실상 공백
'사우나/찜질방/온천' 스캔 결과 전용 유틸 앱 부재:
- **온천갈지도**(5천·4.87, 개인) — 유일 생존자지만 "정적 도감"에 정체. **방문기록 커뮤니티(②)가 없음**, 크래시, velocity≈0(리뷰 3건)
- 사우나플로우(100설치)·온탕(10설치) = 사실상 죽음
- 나머지는 게임(어비스리움)·예약(여기어때)·무관

## 3. 하루검증 — make-or-break 2개 (통과)

### (A) 콘텐츠 총량: 공공데이터로 시드 가능 ✅
- 전국목욕업소표준데이터 17,757행 → **영업중 5,652곳** (폐업 68% 제거)
- 업태 세분류 제공: 목욕탕/사우나 3,941 · 찜질방 1,141 · 한증막 132 · 기타 438
- **사우나(발한실) 보유 2,536**
- ETL 완료(`bathhouse` 시드): 위경도 87% 확보, 변환 오류 0, 지번주소 99.8%, 중복 2건 = **A급**
- 온천은 별도(505곳, 좌표·수질 없어 수작업) — 본체는 목욕/사우나/찜질로 시작
- 상세: [데이터-품질.md](데이터-품질.md)

### (B) 수요·문화: 저빈도 함정 넘음 ✅
- '사우나' 언급량 1.8배 급증(2024.9 9.4만 → 2025.1 16.8만)
- 온천이용자 2024년 5,909만명(+25.4%)
- '리커버리노믹스' — 2030 회복 소비, 2026 메가트렌드 '성장→회복', '소셜 사우나' 확산
- **핵심 조건: "온천만" 아니라 "사우나 중심 온욕 전반"으로 가야 (A)(B) 둘 다 강해짐.** 온천은 진입점, 시장은 온욕 전반.

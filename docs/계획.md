# 사우나 — 구현 계획

전국 온욕(사우나·찜질방·온천) 시설 **지도·검색 + 방문기록 커뮤니티** 앱. 일본 사우나이키타이(サウナイキタイ) 모델의 한국판.

- 앱 ID: `com.jeong.sauna` · 표시이름: 사우나(임시)
- 근거: [계획-근거.md](계획-근거.md) · 스키마: [스키마.md](스키마.md)

## 제품 전략
- **앱(본체)**: 지도·시설검색 + 방문기록/찜/시설평(UGC 엔진). 리텐션·수익의 축.
- **웹(퍼널)**: 시설 5,652곳을 SEO로 색인 → "○○ 사우나/노천탕" 검색 유입 → 정보 보여주고 → **쓰기(기록·전체후기·찜)는 앱으로 유도.** 웹은 읽기 중심 얇게.
- 핵심 자산: 공공데이터 시드 `bathhouses.json`(영업중 5,652곳, 위경도 87%).

## 디자인·컨셉
- **톤 = 따뜻·위트** (동네 목욕탕 정겨움 + MZ 감각. 차가운 프리미엄·과한 캐릭터 회피). 말투 예: "오늘 어디서 뜨끈하게?"
- **모티프 = `~`** (증기·물결 + 개운한 눈). 로고·아이콘·배경 패턴에 반복.
- **색** = 뜨끈한 온기(테라코타·코랄·황톳빛) 주조 + 물/증기 청량(soft teal) 보조 + 웜 뉴트럴 배경. 다크=밤 사우나.
- **UX 원칙** = 정보밀도는 사우나이키타이처럼 깔끔·스캔, 감성은 따뜻(온천갈지도가 못한 지점).

### ~온도 그룹 (게이미피케이션 = 리텐션 엔진)
경쟁 랭킹 아님. **빈도 기반 "온도 그룹"** — 최근 4주 방문일 수로 내 `~` 색이 정해짐(정체성·소속감). 안 가면 창이 롤링돼 자연히 식음(벌점 없이 리텐션).

| 최근 4주 방문일 | 그룹 | ~ 색 |
|---|---|---|
| 주 5회+ | 불가마 | 🔴 |
| 주 2~4회 | 열탕 | 🟠 |
| 주 1회 | 온탕 | 🟡 |
| 격주 | 미온탕 | 🟢 |
| 월 1회 이하 | 냉탕 | 🔵 |

- **어뷰징 방지**: 빈도 = 방문 "날짜" 수(하루 여러 기록도 1일). 업장 붙은 방문기록만, 좋아요·자유글 제외.
- **MVP**: 최근 28일 방문일 수 → 구간 매핑(조회 시 계산 or profile 캐시). 복잡한 공식·식음감쇠는 나중.
- 확장(나중): 그룹별 피드·뱃지·"열탕 진입!" 알림. (구간·창 크기·그룹명은 조정 가능)

## 기술 스택
| 레이어 | 선택 | 이유 |

| 로그인 | **Supabase Auth — Google + Apple** | Apple은 소셜로그인 제공 시 iOS 스토어 필수. 둘 다 Supabase 네이티브 OAuth. 카카오는 추후 옵션(커스텀 OIDC) |
| 배포 | 웹=Vercel, 앱=빌드머신/EAS → 스토어 | — |
| 빌드 | 별도 빌드 머신(이 맥은 계획·코드·웹·Supabase 작업) | 로컬 iOS 빌드 안 함(디스크) |

- 폴더: `app/`(Expo) · `web/`(Next) · 같은 Supabase 공유. 타입은 `supabase gen types`로 생성 공유(모노레포는 과함).

## 티저 = 페이크 도어 (fake door)
미래 기능은 "준비중 탭"으로 죽여두지 말고, **신호를 잡는 티저**로만 남긴다.
- **누르면 탭 카운트 로깅(+선택적 "알림받기")** → 공짜 수요조사("매거진 300탭 → 만들 가치"). 만들기 전 수요 측정 철학과 동일.
- **개수 1~2개, 위치는 "마이/더보기" 안쪽** (메인 탭바는 지도·기록 핵심만).
- **진짜 미래 기능(매거진·굿즈샵)에만.** 커뮤니티(글·리뷰)는 티저가 아니라 **MVP 본체**(posts/reviews).
- 금지: 눌러도 "기대해주세요"만 뜨는 죽은 탭(미완성·신뢰 저하).

## 커뮤니티 & 모더레이션
- **피드 = posts(방문기록+자유)** + **reviews(별점 평가)** 별도. 업장 붙이면 방문기록, 안 붙이면 자유(topic 자동).
- MVP: 좋아요 + **신고(스토어 UGC 필수, 5건 자동차단)** + 유저 차단(Apple 요구) + **EULA 무관용 정책**. 댓글·팔로우는 다음.

## 리스크 & 대응
- **콜드스타트(초기 글 0)** → 공공데이터로 시설 뼈대(5,652곳)가 정보 유틸로 채움 + 본인·지인 부트스트랩 + "첫 기록" empty state. **스크래핑·외부 후기 링크 안 함**(ToS·저작권). 트래션 후 인플루언서.
- **인플루언서 직접 앱화** → 경쟁 전 파트너로 포섭
